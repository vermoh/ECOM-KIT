# Product Vision

Система — это SaaS платформа с управляющим слоем (Control Plane) и набором подключаемых моносервисов (Service Plane).

Цель:
Автоматизировать обогащение товарных данных (SKU) через AI, сохраняя контроль, воспроизводимость и управляемость.

Ключевая ценность:
- быстрое создание структуры характеристик товаров
- автоматическое заполнение данных
- контроль качества через human-in-the-loop
- масштабирование на множество бизнес-доменов

Основной сценарий:
1. Пользователь загружает CSV
2. Система предлагает структуру характеристик
3. Пользователь подтверждает
4. Система обогащает данные
5. Пользователь решает коллизии
6. Экспорт готового результата

Принципы:
- строгая изоляция организаций (tenant isolation)
- контроль доступа через RBAC
- отсутствие “магии” без контроля пользователя
- асинхронная обработка тяжёлых операций
