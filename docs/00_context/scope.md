# Scope

IN SCOPE:
- мультитенантная платформа
- управление пользователями и доступами
- подключение моносервисов
- CSV обработка товаров
- генерация схем характеристик
- enrichment pipeline
- обработка коллизий
- экспорт данных

OUT OF SCOPE:
- полноценный PIM (Product Information Management)
- realtime streaming обработка
- сложный биллинг (пока)
- marketplace интеграции
