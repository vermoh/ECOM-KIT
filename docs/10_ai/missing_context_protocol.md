# Missing Context Protocol

Если для задачи не хватает контекста:
1. Не придумывать новое архитектурное решение.
2. Указать, какого именно документа или правила не хватает.
3. Предложить минимальный doc fragment или ADR draft.
4. Только после этого продолжить реализацию.

Принцип:
better explicit gap than implicit hallucination.
